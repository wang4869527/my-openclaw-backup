[Tue 2026-03-31 15:25 GMT+8] 哪种不容易限流
assistant: **工作区技能 / 手动安装最保险**，完全不走外部接口。

| 方式 | 限流风险 | 原因 |
|------|----------|------|
| **工作区技能（手动复制到 `workspace/skills/`）** | 🟢 无 | 纯本地文件，不调任何 API |
| **npm install** | 🟡 极低 | npm 限流门槛高，日常几乎碰不到 |
| **openclaw skills install** | 🔴 中高 | 底层走 ClawHub，有明确的每分钟请求限制（`429 Too Many Requests`），我们之前遇到过 |

**经验教训：** 之前尝试安装 `free-models-for-openclaw` 时就被 ClawHub 限流（`429` 错误），导致安装中断。

**推荐做法：**
1. 重要技能 → 手动下载 SKILL.md 和相关文件放到 `workspace/skills/<名称>/`
2. 日常安装 → `npm install` 优先
3. `openclaw skills install` → 方便但有概率限流，重试即可
